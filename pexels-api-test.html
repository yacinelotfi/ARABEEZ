About This Page

This page demonstrates how to call the Pexels API and handle the response. It provides examples of both using the fetchPexelsImages function and using the Pexels client directly.

Open the browser console (F12) to see the detailed API response logs.

API Response Structure

The Pexels API returns a response with the following structure:

Example 1: Using fetchPexelsImages

// Call the fetchPexelsImages function with a search term fetchPexelsImages('nature') .then(images => { console.log('Images returned:', images); // Process the images array here }) .catch(error => { console.error('Error fetching images:', error); });

Click "Run Example 1" to fetch and display an image

Example 2: Using the Pexels Client Directly

// Use the Pexels client directly client.photos.search({ query: 'animals', per_page: 5 }) .then(response => { console.log('Full API response:', response); // Process the full response object const photos = response.photos; // Do something with the photos }) .catch(error => { console.error('Error:', error); });

Click "Run Example 2" to fetch and display an image
